 **X 型四轴无人机**（也叫“X 架”）中，电机的编号和位置常按如下方式安排（面向无人机前方来看）：

✅ 电机编号与位置（面向前方,俯视）：

        ↑ 前方（X+）
        
       M1      M2
         \    /
          \  /
           ⊕（机体中心）
          /  \
         /    \
       M3      M4
       
        ← Y+（左）
| 电机编号 | 位置               | 推荐旋转方向     | 常见用途       |
| -------- | ------------------ | ---------------- | -------------- |
| M1       | 左前 (Front-Left)  | **顺时针 (CW)**  | 与 M4 互为对角 |
| M2       | 右前 (Front-Right) | **逆时针 (CCW)** | 与 M3 互为对角 |
| M3       | 左后 (Back-Left)   | **逆时针 (CCW)** | 与 M2 互为对角 |
| M4       | 右后 (Back-Right)  | **顺时针 (CW)**  | 与 M1 互为对角 |
